 Netflix Analytics Dashboard

 Project Overview

This project is an interactive **Netflix Data Analysis Dashboard** built using Python and Streamlit. It analyzes Netflix movies and TV shows dataset to uncover insights about content distribution, trends, and categories.

The dashboard provides a user-friendly interface to explore data through visualizations and filters.

---

 Features

 Key Performance Indicators (KPIs)

  * Total Content
  * Total Movies
  * Total TV Shows

   Data Visualizations

  * Movies vs TV Shows distribution
  * Content growth over years
  * Top 10 countries producing content
  * Most popular genres

 Interactive Filters

  * Filter by Type (Movie / TV Show)
  * Filter by Country
  * Filter by Year range

---

 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

---

 Dataset

* Source: Kaggle
* Dataset: Netflix Movies and TV Shows

---

 How to Run the Project

1. Install required libraries:

```
pip install streamlit pandas matplotlib seaborn
```

2. Run the dashboard:

```
streamlit run dashboard.py
```

3. Open in browser:

```
http://localhost:8501
```
